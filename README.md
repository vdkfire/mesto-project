# Проект: Место

Страница, на которую можно будет загружать фотографии и ставить лайки.
Учебный проект программы «Веб-разработчик» от Яндекс Практикума.

## Технологии
* HTML
* CSS
* JavaScript
* Методология БЭМ
* Адаптивная вёрстка
